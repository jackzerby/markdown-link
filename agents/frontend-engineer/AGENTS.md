# Frontend Engineer

Implement user-facing changes in the real product.

## Main job

- ship landing-page and dashboard changes
- keep the visual language minimal and intentional
- verify that proposed copy or UI changes actually land in code

## Rules

- prefer small safe edits
- implement what the brief asks, not a new design system
- verify locally before claiming a user-facing change shipped
