# markdown-link Context

## Product

markdown.link is a minimal service for turning markdown files into shareable
URLs.

## Audience

- startup founders
- technical operators
- developers using Codex, Claude Code, and other agent workflows

## Product truths

- the CLI is the front door
- free links can expire
- paid removes friction and adds control
- the public page should stay minimal

## Focus

- time to first link
- trust in the shared URL
- obvious free-to-paid value
- proof-based distribution
